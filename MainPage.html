World Happiness report

About the Visualization:For this Visualization, I have used the world happiness report for the year 2016 from Kaggle. The data represents the happiness score estimate along with the extent to which factors like economic production, social support, life expectancy, freedom, absence of corruption, and generosity contribute to making life evaluations higher in each country than they are in Dystopia. This project is based on hybrid interactive slide show. Guidance is provided on each slide to the users that they can see details like annotations, tooltips, sort etc., that they might be interested on or they can choose just to continue.

Scenes:Each scene is a single chart with a specific goal related to the same subject and in a specific order to best convey the message. Each slide has similar template, fonts, hue and tool tip to provide visual consistency on scene.

Annotations:Annotations are provided as required and are highlighted in blue color to show the details. The legend and labels along the X and Y axis provides additional annotations for the chart.

Triggers:The previous and next buttons, page numbers in the navigation bar of each page acts a triggers and sort checkbox on all bar charts also acts as one of the triggers. The mouse click acts as a trigger event for navigation. The mouse hover is another user interface event. The hover event triggers the display of tooltip in all the line and bar charts.

Parameters:In bar graphs, check box entry acts as the parameter for the graph, user can sort and unsort the data. Slide numbers, current page highlighted also serve as parameters. Based on the user sort parameter selection, appropriate data is filtered and respective data is displayed.
